## 5. Требования к разделам проектной документации

### 5.1 АР — Архитектурные решения
- Планы этажей (типовой, технические, кровля), фасады, разрезы; экспликации помещений
- Узлы: примыкания ограждений, витражей/окон, парапеты, сопряжения кровли и фасада
- Фасады: навесная вентилируемая система с негорючей теплоизоляцией; оконные блоки ПВХ/алюминий с двухкамерным стеклопакетом (низкоэмисс.)
- Входные группы, МГН: пандусы/лифты, тактильные индикаторы; ширины дверей ≥ 900 мм в МОП
- Помещения 1 этажа: встроенно‑пристроенные (ретейл/сервисы) — без ухудшения условий эвакуации
- Расчёты: теплотехника ограждений (СП 50), инсоляция и КЕО (СП 52), при необходимости акустика
- Выдача: графика (PDF/DWG), модель (IFC4/RVT), спецификации отделки, ведомости заполнений проёмов

### 5.2 КР — Конструктивные решения
- Система: монолитный железобетонный каркас с ядрами жёсткости; перекрытия монолитные
- Фундаменты: плитный фундамент (толщина предварительно 800–1000 мм) с ростверком под ядра; гидроизоляция обмазочная + оклеечная
- Материалы: бетон классов B25–B30, арматура A500C; защитные слои: 25–35 мм (по эксплуатации)
- Перекрытия: толщина 200–220 мм (уточняется расчётом); лестничные марши монолитные
- Расчёты: по СП 20/63, моделирование в SCAD/LIRA/ETABS; нагрузки — по СП 20 для региона строительства
- Узлы: армирование, закладные детали под фасады, отверстия под инженерные системы
- Выдача: комплект КР (PDF/DWG), расчётное обоснование, спецификации, модель (IFC4/RVT)

### 5.3 ОВ — Отопление, вентиляция и кондиционирование
- Источник тепла: центральное теплоснабжение через ИТП; параметры отопления 95/70 °C (уточняются ТУ)
- Отопление: двухтрубная система с нижней разводкой; приборы — стальные панельные радиаторы с ТРВ
- Вентиляция: жилые квартиры — вытяжка естественная через шахты из кухонь/санузлов; подпитка — приток через оконные клапаны/щели
- МОП/помещения 1 этажа — ПВУ с рекуперацией; подземный паркинг — механическая приточно‑вытяжная с СО‑датчиками
- Противодымная защита: системы подпора в лифтовых холлах и Н1 лестницах; дымоудаление из коридоров (СП 7)
- Кондиционирование: при необходимости — сплит‑системы с размещением наружных блоков на кровле/технических площадках
- АСУ/Диспетчеризация: узлы автоматики ИТП и вентиляции с подключением к диспетчерской
- Выдача: планы, схемы, аксонометрии, спецификации, теплотехнические и аэродинамические расчёты, модель (IFC4)

### 5.4 ВК — Внутренний водопровод и канализация
- ХВС/ГВС: централизованные; ГВС с рециркуляцией; узлы учёта в ИТП; термостатические клапаны на стояках
- Канализация: хозяйственно‑бытовая — раздельные стояки; ливневая/дренажная — для стилобата и кровли с воронками
- Защита от легионеллы: поддержание t ГВС 60±5 °C, промывки, рециркуляция; материалы труб — PPR/PEX/сталь (по зоне)
- Водоразбор: приборы по типовым спецификациям; фильтрация/умягчение по результатам анализа воды
- Выдача: планы, аксонометрии, гидравлические расчёты, спецификации, модель (IFC4)

### 5.5 ТС — Теплоснабжение (ИТП/присоединение)
- Присоединение к теплосети по независимой схеме; учет тепла на вводе
- ИТП: пластинчатые теплообменники (отопление/ГВС), насосные группы, узлы смешения, расширительные баки, КИПиА
- Параметры систем: отопление 95/70 °C; П1/П2 для вентиляции 60/40 °C (уточнить по ТУ)
- Теплоизоляция трубопроводов; мероприятия по коррозионной защите и конденсату
- Выдача: принципиальные схемы, планы помещений ИТП, спецификации, тепловой баланс, модель (IFC4)

## 6. BIM/координация
- Координатная система: локальная (проектная) с закреплением нулевой точки и осей; трансформация относительно ГСК — по исполнительной схеме геодезии
- Уровни и оси: единые для всех разделов; шаблон уровней и имён предоставляется
- LOD/LOI: П — 200–250; Р — 300–350; РД — 350–400; обязательные параметры в объектах по классификатору проекта
- Форматы обмена: RVT (основной), IFC4 (Coordination/Reference View), DWG (линейка), PDF (выпуск), XLSX (ведомости)
- Координация: еженедельные clash‑сессии; устранение коллизий ≤ 5 раб. дней; протоколирование RFI

## 7. Состав и формат выдачи
- Пояснительная записка, расчёты, графика (PDF), исходники (DWG/RVT), IFC4
- Спецификации, ведомости объёмов, ведомости отделки (АР), экспликации
- Требования к электронным файлам: метрические единицы (мм/м), кодировка UTF‑8, слои/семейства по стандарту проекта
- Нумерация листов и марка чертежей — по стандарту предприятия Заказчика

## 8. Требования по пожарной безопасности и эвакуации
- Функциональный класс Ф1.3, степень огнестойкости I; противопожарные преграды согласно зонированию
- Две незадымляемые лестницы типа Н1 в каждой секции; лифтовые холлы отделены противопожарными перегородками EI 45–60
- Пределы огнестойкости несущих конструкций не ниже R 120; перегородок по назначению не ниже EI 45 (уточняется по проекту)
- Системы противодымной защиты и подпора, дымоудаление из коридоров и паркинга (СП 7); пожарный лифт в каждой секции
- Эвакуационное освещение и СОУЭ 3‑го типа для МОП и паркинга (по заданию ПБ)

## 9. Энергоэффективность и санитария
- Целевой класс энергоэффективности: не ниже B+; узлы учёта энергоресурсов (тепло, вода, электро)
- Ограждающие конструкции: U‑значения в соответствии с СП 50 для региона строительства; теплотехнический расчет обязателен
- Инсоляция/освещённость — соответствие СП 52; акустические требования к ограждающим конструкциям — по заданию Заказчика
- Требования МГН: доступность входов, путей, санитарных узлов и лифтов

## 10. Сроки и этапность
- П: 6 недель с даты утверждения ТЗ и получения полного комплекта исходных данных
- Р: 12 недель после согласования П (или параллельно по согласованию)
- РД: по разделам — 16–24 недели с поквартальной выдачей комплектов
- Промежуточные сдачи: каждые 2 недели — отчёт и координационная модель

## 11. Порядок согласования и изменения
- Изменения ТЗ оформляются листом изменений и протоколом согласования сторон
- Все вопросы/коллизии фиксируются в реестре RFI с целевыми сроками ответа (≤ 3 раб. дней)
- Приоритет версий: при расхождениях первичны согласованные PDF и модели IFC соответствующего выпуска
